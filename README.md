# CookAndShare-AplicacionesWebEscalables
“Cook&amp;Share”, es un gestor de recetas de cocina en el cual los usuarios pueden consultar recetas de diferentes categorías/tipos, así como compartir sus propias recetas con la comunidad. El sitio busca crear un espacio en el que las personas interesadas en la cocina puedan encontrar, calificar y comentar recetas de forma fácil y organizada.
